# GameCapabilityAssessment-rockpell

여러가지 미니게임을 조준력, 집중력, 사고력, 순발력, 리듬감과 같은 5가지 항목으로 나누어서 결과에 따라 플레이어의 게임능력을 평가하는 게임입니다.

스마일게이트 멤버십(예비 창업자를 위한 지원 프로그램) 10기에 참가한 프로젝트입니다.

이미지와 사운드의 라이센스 문제로 인해 스크립트만을 별도로 올린 저장소입니다.
